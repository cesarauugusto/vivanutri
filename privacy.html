Política de Privacidade do Vivanutri

Última atualização: 01/06/2025

1. Quais dados coletamos

  • Dados de autenticação: e-mail e nome de usuário (via Firebase Auth ou Google Sign-In).
  • Dados de uso: interações com a interface (ex.: itens salvos, preferências).

2. Como usamos seus dados

  • Para autenticar sua sessão e manter você logado.
  • Para armazenar e sincronizar seus itens entre dispositivos.
  • Para melhorar o app, por meio de análises anônimas de uso.

3. Compartilhamento de dados

Não vendemos, alugamos nem compartilhamos seus dados pessoais com terceiros, exceto quando exigido por lei ou para prestar serviços essenciais (por exemplo, Firebase).

4. Segurança

Utilizamos segurança de ponta (HTTPS, Firebase Security Rules) para proteger seus dados contra acesso não autorizado.

5. Seus direitos

  • Solicitar correção ou exclusão de seus dados pessoais.
  • Revogar seu consentimento a qualquer momento, excluindo sua conta.

6. Contato

Em caso de dúvidas sobre esta política, envie um e-mail para: cesartri2012@gmail.com.
